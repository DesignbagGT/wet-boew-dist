Data Ajax

Un enveloppeur simple pour AjaxLoader qui insère de contenu télécharger via AJAX.

Insérer du contenu après un élément (data-ajax-after)

Exemple

Bonjour tout le monde

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.

Bonjour tout le monde - partie 2

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.

Visualiser le code 
<section data-ajax-after="ajax/data-ajax-extra-fr.txt" class="original">
	<h4 id="anchor1">Bonjour tout le monde</h4>
	<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.</p>
</section>
<section data-ajax-after="ajax/data-ajax-extra-2-fr.txt" class="original">
	<h4 id="anchor1">Bonjour tout le monde - partie 2</h4>
	<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.</p>
</section>
