Correctif «&#160;datalist&#160;» (saisie automatique)

La balise datalist spécifie une liste d'options pré-définis pour un élément input.

Exemple

Visualiser le code 
<label for="city">Ville</label>
<input type="text" id="city" name="city" list="suggestions" />
<datalist id="suggestions">
