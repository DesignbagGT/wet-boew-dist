Exemples pratiques

Nom Catégorie Description Mots clés
Calendrier d'événements Plugiciel Un calendrier servant à consulter une liste d'événements. cal-events
Data Ajax Plugiciel Un enveloppeur simple pour AjaxLoader qui insère de contenu télécharger via AJAX. data-ajax
Data Inview : Bannière Plugiciel Affiche une bannière quand une section se déplace hors de la clôture. data-inview
Data Inview : Panneau Plugiciel Affiche un panneau quand une section se déplace hors de la clôture. data-inview
Data Picture Plugiciel Conversion de Picturefill guidé par l'événement. data-picture
Favoricône Plugiciel Offre la possibilité d'ajouter et de mettre à jour les favoricônes d'une page. favicon
Formulaire de rétroaction Plugiciel Un formulaire normalisé pour soumettre une rétroaction au sujet d'une page ou d'un site Web particulier. feedback
Fils de syndication Plugiciel Affiche les articles d’un ou plus fils de syndication. feeds
Notes de bas de page Plugiciel Une méthode cohérente et facile d’accès pour répertorier les notes de bas de page. footnotes
Validation de formulaires Plugiciel Effectue la validation de formulaires Web selon un ensemble de règles de base avant qu'ils soient soumis. formvalid
Lightbox Plugiciel Crée une galerie photo sur une page Web. lightbox
Menu Plugiciel Une barre de menu interactive comprenant des sous-menus optionnels. menu
Lecteur média Plugiciel Un lecteur média accessible permettant d'intégrer des contenus vidéo et audio sur les pages Web. multimedia
Prettify Plugiciel Affiche de la syntaxe des extraits de code source. prettify
Expiration de la session Plugiciel Ferme la session de l'utilisateur après un certain délai. session-timeout
Gadget de partage Plugiciel Permet à l'utilisateur de partager un contenu Web sur les plateformes de médias sociaux. share
Tables Plugiciel Ajoutes des fonctionnalités aux tableaux telles que la recherche, le tri, le filtrage et la pagination. tables
Interface à onglets Plugiciel Comprime plusieurs sections de contenu dans le même espace physique et fourni des onglets qui permettent à l'utilisateur de choisi la section de contenu à afficher. tabs
Mise en surbrillance de texte Plugiciel Met en surbrillance sur une page Web les mots choisis. texthighlight
Basculer Plugiciel Plugiciel qui permet un lien à basculer des éléments entre les états activé et désactivé. toggle
Chronologie intégrée Twitter Plugiciel Affiche des chronologies intégrées Twitter. twitter

Correctif «meter» (compteur) Correctif Émule l’élément <meter> pour les navigateurs qui ne le supportent pas. meter
Correctif «progress» (barred de progression) Correctif Émule l’élément <progress> pour les navigateurs qui ne le supportent pas. progress
Correctif «input type="range"» (barre coulissante) Correctif Émule l’élément <input type="range"> pour les navigateurs qui ne le supportent pas. slider
